What is this WebApp? This is a web application that lets you create a rommate list and assign them house-hold duties.

How to use: enter each roommate's name, email address and phone number. Create all duties using the Duty Creation form. Each roommate and duty will then appear Assign Duties here section, where you can assign duties and also select which rotation pattern you want to use.
